🖼️ 2. Como Adicionar Imagens (Thumbs)

Onde ficam as imagens: pasta assets/thumbs/ no repositório GitHub

Formato aceito: .jpg ou .svg

Tamanho ideal: 320 × 180 px (proporção 16:9)

Tamanho máximo: 200KB por arquivo

Passo a passo — Adicionar imagem pelo GitHub

1
Acesse o repositório: github.com/ossluz/Dramaluxe
2
Clique na pasta assets → depois em thumbs
3
Clique em "Add file" → "Upload files"
4
Arraste a imagem para a área indicada. O nome do arquivo deve ser exatamente igual ao que está no catálogo. Por exemplo, se o catálogo diz thumb:'assets/thumbs/nirvana-in-fire.jpg', o arquivo deve se chamar nirvana-in-fire.jpg
5
Clique em "Commit changes" → o site atualiza em ~30 segundos

💡 Onde baixar imagens de qualidade:

  • MyDramaList — clique no drama → clique no poster → clique direito → "Salvar imagem como"
  • TMDB — pesquise o drama → clique em "Imagens" → escolha um backdrop (16:9)
  • Google Imagens — pesquise "NOME DO DRAMA" poster official → Ferramentas → Tamanho → Grande

🎞️ 3. Como Adicionar Novos Vídeos ao Catálogo

O catálogo fica no arquivo js/catalog.js no GitHub. Para editar:

1
Acesse github.com/ossluz/Dramaluxe → js/catalog.js
2
Clique no lápis ✏️ no canto superior direito para editar
3
Cole o código do novo item (use os modelos abaixo)
4
Clique em "Commit changes"

Como pegar o ID de um vídeo do YouTube

Na URL do YouTube: https://www.youtube.com/watch?v=Bsa_Ur3m0hA

O ID é a parte após v= — neste exemplo: Bsa_Ur3m0hA

Para playlists de episódios, pegue o ID de cada vídeo individual (cada episódio).

📺 4. Modelo Pronto — Nova Série

Cole este modelo dentro do array series: [ em catalog.js, junto com as outras séries:

/* Cole dentro de  series: [ ... ]  */
{
  key:'minha-nova-serie',              // ID único, sem espaços, use hífen
  title:'Nome em Português',
  titleOriginal:'Nome Original / 标题',
  region:'c-drama',                      // c-drama | k-drama | j-drama | thai-drama
  genre:'romance',                     // romance | fantasy | action | comedy | drama
  audio:'dubbed',                      // dubbed = dublado | subbed = legendado
  year:2024,
  cast:'Ator Principal, Atriz Principal',
  rating:'8.5',
  featured:false,                      // true = aparece no banner, false = só nos cards
  desc:'Descrição do drama em português. Máximo 2 frases. Informe se é dublado ou legendado no final.',
  thumb:'assets/thumbs/minha-nova-serie.jpg', // nome do arquivo de imagem
  episodes:[
    'ID_EP1_YOUTUBE',   // ID do episódio 1
    'ID_EP2_YOUTUBE',   // ID do episódio 2
    'ID_EP3_YOUTUBE',   // ... continue para todos os episódios
  ],
},

🚀 6. Como Publicar no GitHub

1
Acesse github.com/ossluz/Dramaluxe
2
Para editar o catálogo: clique em js → catalog.js → ícone de lápis ✏️
3
Para adicionar imagens: clique em assets → thumbs → Add file → Upload files
4
Após qualquer alteração, clique em "Commit changes"
5
Aguarde ~30 segundos e acesse ossluz.github.io/Dramaluxe/ para ver as mudanças

✅ 7. Checklist Rápido — Ao Adicionar Novo Conteúdo

Para cada novo drama/filme que você quiser adicionar:

  • ☐ Verificar se é romance heterossexual ou ação/artes marciais
  • ☐ Verificar se tem áudio em PT-BR (dublado ou legendado)
  • ☐ Copiar o modelo de série ou filme acima
  • ☐ Preencher todos os campos (key, title, region, genre, audio, year, cast, rating, desc, thumb)
  • ☐ Pegar os IDs dos episódios/filme no YouTube
  • ☐ Baixar a imagem do poster (320×180px) e nomeá-la igual ao campo thumb
  • ☐ Subir a imagem em assets/thumbs/ no GitHub
  • ☐ Colar o código no catalog.js e commitar

⚠️ Regras importantes:

  • O campo key deve ser único — nenhum outro drama pode ter o mesmo
  • Sempre termine o bloco com }, (vírgula no final)
  • Use apenas aspas simples 'assim', nunca duplas "assim"
  • O nome do arquivo de imagem deve bater exatamente com o campo thumb
  • Para year e featured — sem aspas
